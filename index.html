What To Do Next

1Update fixtures.json weekly. This drives weather by match and venue. Without real fixtures, Weather Hub is rubbish.
2Run Update Weather Data. GitHub → Actions → Update Weather Data → Run workflow.
3Press Refresh Latest Data in the app. This loads weather.json, players.json and my_team.json into the browser.
4Keep players.json accurate. Ownership, BE, captain %, VC %, injuries and projections are the real edge.
5Use Weekly Trades, not Clear Team. Clear Team now has confirmation, but weekly trades are the correct way to manage the season.

Season Reset Checklist

New season drawReplace fixtures.json with the official NRL draw for the season.
New players/pricesReplace/update players.json with current season positions, prices, averages and teams.
Bye roundsCheck all club bye rounds and Origin-affected rounds.
VenuesUpdate venues.json if stadium names or locations change.
Your teamUpdate my_team.json or reload via app if your squad changes.
AutomationCheck both GitHub Actions: Update SuperCoach Data and Update Weather Data.

# Joel SuperCoach real team files

Upload/replace these in your GitHub repo root:

- `players.json`
- `my_team.json`
- `index.html`  (optional but recommended, because it auto-loads `my_team.json`)

What this does:
- `players.json` now contains the players identified from Joel's screenshots.
- `my_team.json` contains Joel's actual squad list, captain, vice-captain and flex.
- The patched `index.html` auto-loads `my_team.json` only if there is no saved team already in browser storage.

Important:
If your browser already saved an old/empty team, open the app > My Team > Clear Team, then refresh.
